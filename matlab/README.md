# autd3-matlab

Version 8.5.0

This is a sample program to drive [autd3](https://github.com/shinolab/autd3) in MATLAB.

## Build

Before running example, please run `build.ps1` on Windows or `build.sh` on mac and Linux.

## Example

Run `soem` or `simulator` in MATLAB.
