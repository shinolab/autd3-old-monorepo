# Gain

AUTDは各振動子の位相/振幅を個別に制御することができ, これによって様々な音場を生成できる.
`Gain`はこれを管理するクラスであり, SDKにはデフォルトでいくつかの種類の音場を生成するための`Gain`がデフォルトでいくつか用意されている.

- [Null](./gain/null.md)
- [Focus](./gain/focus.md)
- [Bessel](./gain/bessel.md)
- [Plane](./gain/plane.md)
- [Grouped](./gain/grouped.md)
- [Holo](./gain/holo.md)
- [Cache](./gain/cache.md)
