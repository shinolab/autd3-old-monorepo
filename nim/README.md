# autd3_nim

version 2.6.2

## Run examples

- First, change directory to `examples`, and then run `build.ps1` (Windows) or `build.sh` (Linux, macOS) to build native libraries.

  ```
  cd examples
  pwsh build.ps1
  ```

  or

  ```
  cd examples
  ./build.sh
  ```

- And then, run examples

  ```
  nimble run soem
  ```

  or

  ```
  nimble run simulator
  ```
